## Based on elm-webpack-starter-kid

Starter of bare minimum of code. Up-to-date dependencies.

# How to run
```
git clone https://github.com/wiewioraseb/tele-address-base.git
cd client-side
npm i \
npm start

In a web browser navigate to http://localhost:8181
```
